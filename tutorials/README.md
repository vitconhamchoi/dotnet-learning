# Distributed .NET Tutorials

Bộ tài liệu này gồm 7 tutorial chuyên sâu, mỗi bài tập trung vào một thư viện quan trọng trong hệ .NET distributed application:

1. Orleans
2. MassTransit
3. Marten
4. Wolverine
5. Dapr
6. Hot Chocolate
7. .NET Aspire

Mỗi bài được viết theo hướng thực chiến:
- giải thích khái niệm cốt lõi
- chỉ ra khi nào nên dùng và không nên dùng
- có code cụ thể
- đi qua các tính năng quan trọng nhất của thư viện
- có demo flow hoặc kiến trúc mẫu
